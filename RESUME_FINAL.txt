## 📊 NOUVELLES APIs HTTP

### GET /health
Vérification simple - le service tourne-t-il?
```bash
curl http://localhost:3001/health
```

### GET /api/whatsapp/status
Détails complets de l'état
```bash
curl http://localhost:3001/api/whatsapp/status
```

### GET /api/whatsapp/qr-status
Y a-t-il un QR code à scanner?
```bash
curl http://localhost:3001/api/whatsapp/qr-status
```

### POST /api/whatsapp/restart
Redémarrer le service
```bash
curl -X POST http://localhost:3001/api/whatsapp/restart
```
