# 📋 GUIDE DE TEST NÉOBOT POUR CLIENTS

## 🎯 OBJECTIF
Tester le système avec de vrais clients pendant 2-3 jours

## 👥 CLIENTS À CONTACTER
- [ ] 1 Restaurant/snack
- [ ] 1 Boutique de vêtements  
- [ ] 1 Service (coiffure, esthétique)
- [ ] 1 Vendeur en ligne

## 💬 SCÉNARIOS DE TEST
**Demander aux clients d'envoyer ces messages :**

### Pour restaurants :
1. "Bonjour, vous avez le menu ?"
2. "Combien le ndolé ?"
3. "Je veux du poulet braisé avec alloco"
4. "Vous livrez à [leur quartier] ?"
5. "C pour une commande maintenant"

### Pour boutiques :
1. "Salut, vous avez des [produits] ?"
2. "C'est combien le [produit] ?" 
3. "Taille [taille] disponible ?"
4. "Je veux commander"
5. "Livraison gratuite ?"

## 📊 MÉTRIQUES À NOTER
- [ ] Temps de réponse
- [ ] Compréhension des messages
- [ ] Pertinence des réponses
- [ ] Problèmes rencontrés
- [ ] Suggestions d'amélioration
